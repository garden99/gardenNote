# 每日常用

> 以下
> 
> AA=Alt
> 
> SS=Shift
> 
> CC=ctrl
> 
> EE=Enter
> 
> BB=Backspace
> 
> && = 自定义

### &&  CC+AA +D 将下一个匹配项目加入到选择

### AA+insert 生成方法如 get/set toString

### AA + J 将下一个匹配项加入光标

### SS + EE 直接转到下一行

### CC + W 扩选

### CC + AA + L 格式化
